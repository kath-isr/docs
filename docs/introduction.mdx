# Welcome to InStore Reviews

InStore Reviews is a comprehensive QR-based customer feedback system that enables businesses to collect, manage, and analyze customer feedback in real-time. The system features instant SMS notifications for negative feedback and social media integration for positive reviews.

## System Overview

Our application is built with:
- **Backend**: Laravel PHP framework
- **Frontend**: Vue 3 with Composition API
- **Database**: MySQL
- **Local Development**: XAMPP

## Key Features

### 1. QR Code System
- Dynamic QR code generation
- Custom QR code templates
- Location-specific tracking

### 2. Real-time Notifications
- Instant SMS alerts for negative feedback
- Email notifications
- Dashboard alerts

### 3. Social Media Integration
- Automatic social media sharing
- Review aggregation
- Social media analytics

### 4. Analytics Dashboard
- Real-time feedback monitoring
- Trend analysis
- Custom reporting
