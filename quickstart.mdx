## Getting Started

This guide will help you get up and running quickly with our platform.

### Installation

<AccordionGroup>
  <Accordion icon="download" title="System Requirements">
    Before you begin, make sure you have:
    - Node.js 18 or higher installed
    - npm or yarn package manager
    - Git for version control
  </Accordion>
  <Accordion icon="rectangle-terminal" title="Installation Steps">
    1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/yourproject
    ```

    2. Install dependencies:
    ```bash
    npm install
    ```

    3. Start the development server:
    ```bash
    npm run dev
    ```
  </Accordion>
</AccordionGroup>
