## Installation

### 1. Backend Setup

First, set up the Laravel backend:

```bash
# Install PHP dependencies
composer install

# Create environment file
cp .env.example .env

# Generate application key
php artisan key:generate

# Run database migrations and seeders
php artisan migrate:fresh --seed

# Start Laravel development server
php artisan serve
```

### 2. Frontend Setup

In a new terminal, set up the Vue.js frontend:

```bash
# Install npm dependencies
npm install

# Start development server
npm run dev
```

## Key Features

- **Authentication**: Secure user authentication system
- **Review Management**: Create and manage store reviews
- **QR Code System**: Generate and manage QR codes for stores
- **Location Services**: Google Maps integration for store locations
- **Phone Verification**: SMS verification via Twilio
- **Image Processing**: Handle store images and QR codes
